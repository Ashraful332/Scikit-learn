# Scikit-learn

## start project
To start the venv 
```bash
python3 -m venv venv
source venv/bin/activate
```

Install packeg
```bash
pip install packeg-name
```

To of the venv
```bash
deactivate
```

to see all install packages
```bash
pip list
```

to create requirement.txt
```bash
pip freeze > requirements.txt
```
